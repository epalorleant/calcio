# Calcio Frontend

React + TypeScript single-page app for managing futsal sessions.

## Layout
- `src/api/` – API client wrappers.
- `src/components/` – Shared UI components.
- `src/pages/` – Route-level pages (sessions, players, admin).
- `src/hooks/` – Reusable hooks.
- `src/types/` – Shared TypeScript types.

## Local run
```bash
npm install
npm run dev
```
